# 如何使用

### setp1
- 把源文件拷到你的  _src/包名_  目录下.

### setp2
- 在布局文件中加入该组件，如下所示： <br>

### setp3
- 在你的代码中引用该组件

```
  <com.feiqishi.customview.ThreeMenu
        android:id="@+id/view"
        android:layout_width="70dp"
        android:layout_height="70dp" >
  </com.feiqishi.customview.ThreeMenu>
```
